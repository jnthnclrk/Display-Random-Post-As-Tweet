== Description ==

A simple plugin to display a random post in a format ready for posting to Twitter.

* Select post types
* Select page to be used for tweet display

== Installation ==

1. Upload `Display-Random-Post-As-Tweet/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on 'Display Random Post As Tweet Settings' in the Settings menu
